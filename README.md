# Ecommerce
Project Title: E-Commerce Website – Front-End Development Project
Description:
This project is a fully functional e-commerce website built using pure HTML, CSS, and JavaScript — no frameworks or libraries used. It was developed as part of a university front-end web development course to demonstrate real-world web application skills.

What the website includes:
The website simulates a complete online clothing store with the following pages and features:
Pages: Homepage, Product Listing, Product Detail, Shopping Cart, Checkout, Order Success/Failed, Login, Sign Up, Forgot/Reset Password, My Account (Orders, Wishlist, Address, Password, Account Details), About, and Contact.
Key Features:

Product browsing with filters by category, color, size, and price range
Product detail page with image gallery, color/size selector, quantity control, and customer reviews
Fully working shopping cart with add, remove, and quantity update
Checkout flow with shipping form and order placement
User authentication system (register, login, logout)
Personal account dashboard with order history and wishlist
Toast notification system for user feedback
Responsive design that works on mobile and desktop

Data Storage:
All data is stored locally in the browser using localStorage — including the cart, wishlist, user session, and order history. No backend or database is required.
Goal:
The goal of this project is to practice and demonstrate skills in semantic HTML structure, CSS layout and design, JavaScript DOM manipulation, event handling, and client-side state management.
